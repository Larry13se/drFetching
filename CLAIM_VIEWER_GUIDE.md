# Medicare Claims Viewer Guide

## What It Does

A beautiful web interface that displays your extracted Medicare claims data in a professional, easy-to-read format - similar to Medicare.gov's design.

## Features

✅ **Modern UI** - Clean, responsive design that works on desktop and mobile  
✅ **Statistics Dashboard** - See totals at a glance (claims, services, amounts)  
✅ **Expandable Cards** - Click any claim to view full details  
✅ **Complete Data** - Shows all claim info, services, and financial summary  
✅ **Color-Coded** - Important amounts highlighted for easy scanning  
✅ **No Server Required** - Runs entirely in your browser (100% client-side)

## How to Use

### Step 1: Extract Data
First, run the extraction script to generate your CSV file:
```bash
python test_detailed_claims_extractor.py
```

This creates: `detailed_claims_test_output.csv`

### Step 2: Open Viewer
Simply double-click: **`claim_viewer.html`**

Your default browser will open the viewer.

### Step 3: Load Data
1. Click **"📂 Choose CSV File"** button
2. Select your CSV file:
   - `detailed_claims_test_output.csv` (test script output)
   - `medicare_results_FINAL.csv` (main processor output)
3. Data loads instantly!

### Step 4: Explore Claims
- **Summary cards** at top show totals
- **Claim cards** show provider and amount you owe
- **Click any card** to expand and see:
  - Full claim information
  - All services with procedure codes
  - Complete financial breakdown

## What You'll See

### Dashboard Stats
- Total number of claims
- Total number of services
- Total amount charged by providers
- Total you may be billed

### Each Claim Shows
**Basic Information:**
- Date of service
- Provider name and address
- Claim processed date
- Claim number

**Services Provided:**
- Service name and procedure code
- Quantity
- Provider charged amount
- Medicare approved amount
- Applied to deductible
- Coinsurance
- You may be billed

**Financial Summary:**
- Total amount charged
- Total Medicare approved
- Applied to deductible
- Total coinsurance
- **Total you may be billed** (highlighted)
- Medicare paid provider/you
- Blood deductible
- Physical therapy, psychiatric, occupational therapy charges
- Patient paid provider

## Tips

💡 **Multiple Files**: You can load different CSV files without refreshing - just click the upload button again

💡 **Sorting**: Claims are displayed in the order they appear in your CSV (usually newest first)

💡 **Mobile Friendly**: The viewer automatically adjusts for smaller screens

💡 **No Internet Required**: Once opened, works completely offline

## Troubleshooting

**"Could not find complete_claim_json column"**
- Make sure you're loading the correct CSV file
- The file should be generated by `test_detailed_claims_extractor.py`

**"No valid claims found in CSV"**
- Check that the CSV has data rows (not just headers)
- Open CSV in a text editor to verify the `complete_claim_json` column has JSON data

**Nothing happens when I click upload**
- Make sure you selected a `.csv` file
- Try refreshing the page and uploading again
